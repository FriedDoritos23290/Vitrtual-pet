# Виртуальный Питомец 

**Virtual Pet** — это интерактивная игра в стиле, написанная на Python с использованием библиотеки Pygame. Ухаживайте за своей виртуальной собакой: кормите её, одевайте, играйте в мини-игры и зарабатывайте монеты, чтобы улучшать её жизнь!

---

## Основные возможности

-  **Кормление**: покупайте и скармливайте еду для восстановления сытости и здоровья.
-  **Одежда**: покупайте и надевайте стильную одежду для вашей собаки.
-  **Мини-игра**: ловите игрушки, чтобы повысить уровень счастья.
-  **Экономика**: зарабатывайте монеты кликами и автоматически, улучшайте доход.
-  **Прогресс**: все данные сохраняются между сеансами в `save.json`.
-  **Game Over**: если один из параметров (здоровье, сытость, счастье) упадёт до нуля — игра завершится.

---

## Структура проекта

```
virtual-pet/
├── images/               # Все изображения (фон, иконки, еда, одежда и т.д.)
├── save.json             # Файл сохранения прогресса
├── main.py               # Основной файл игры (ваш код)
└── README.md             # Этот файл
```

> Чтобы игра запустилась, необходимо создать структуру папок `images/` и поместить туда все нужные изображения согласно путям в коде.

---

## Требования

- Python 3.7+
- Pygame

Установите зависимости:

```bash
pip install pygame
```

---

## Запуск игры

1. Убедитесь, что структура папок `images/` соответствует ожиданиям кода:
   - `images/background.png`
   - `images/dog.png`
   - `images/menu/...`
   - `images/food/...`
   - `images/items/...`
   - `images/toys/...`
2. Запустите игру:

```bash
python main.py
```

---

## Управление

- **ЛКМ** — клик для получения монет (+ автоматический доход раз в секунду)
- **Кнопки в интерфейсе** — открывают меню еды, одежды, мини-игр
- **A / D** — управление собакой в мини-игре (влево/вправо)
- **ESC** — возврат в главное меню из любого подменю

---

## Сохранение

Игра автоматически сохраняет прогресс при закрытии окна.  
При "Game Over" сохранение сбрасывается к начальным значениям.
